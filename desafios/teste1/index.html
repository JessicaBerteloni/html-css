Dominio de hospedagem

Dominio= Nome do servidor ex... (Cursoemvideo.com)

Url= Uniform resource Locator ex... (www / http ou https://)

Tld= Top Level Domail ex... (pais)

Gtld= São tlds genericos, sem indicação de pais ex... (.com/.net/.gov/.org/.io/.info/.online/.store etc)

Cctld= São tlds com designação de pais (.com.br/edu.us/.co.fr/.jp/.es etc)

Sub Dominio= Site pricipal (Www.google.com.br/ imagens.google.com.br/ mapas.google.com.br/ mail.google.com.br)

Codigos

HEAD= Cabeça do site

TITLE= Titulo da pagina

BODY= Corpo da pagina, ou seja, todo o resto do site onde estarão os seus textos

P= Criar paragrafos e separa-los de forma correta e organizada

H1= Nome dado para titulos que ficarão dentro da pagina que ão de h1 até h6

HR= Quebra de linha

META= Define qualquer informação de metadados que não podem ser definidos por outro elementos HTML
